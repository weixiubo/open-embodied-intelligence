# 演示排练清单

## 启动前

1. 确认 `.env` 已配置。
2. 如果接真机，先检查串口权限和设备路径。
3. 先用 `sim` 模式排练一轮，再切到 `auto` 或 `real`。

## 推荐演示顺序

1. 启动系统，展示启动自检和当前模式。
2. 说 `机器人状态`，让系统播报当前状态。
3. 说 `动作列表`，展示动作库已加载。
4. 说 `跳舞10秒`，展示命令确认。
5. 确认后开始跳舞。
6. 说 `停止跳舞`，展示即时停止。
7. 做一轮简单 AI 对话。
8. 如果硬件不可用，切到 `sim` 模式继续演示。

## 故障补救话术

- 语音没听清：重新说一遍完整命令，例如 `跳舞10秒`。
- 串口掉线：系统会切到模拟模式，继续展示动作编排和语音交互。
- 网络抖动：AI 对话会降级为固定反馈，跳舞主流程不受影响。
